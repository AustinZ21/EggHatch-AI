EggHatch-AI

A lightweight agentic recommendation system for PC building and gaming laptop shopping, grounded in product data, review signals, and multi-turn context.

Workflow

From shopper intent to structured recommendation signals.

EggHatch-AI breaks a shopping question into a small set of specialist steps before synthesizing an answer through a local LLM.

01

Understand

Parse user intent, budget, use case, and follow-up context.

02

Load

Read local product and review fixtures for gaming laptop analysis.

03

Analyze

Extract sentiment, review themes, and recurring product concerns.

04

Rank

Surface candidate products using filters, ratings, and feature signals.

05

Synthesize

Generate a conversational recommendation with explainable context.

Scope

Small project, real agentic product thinking.

The repository is intentionally compact: a Streamlit demo, local fixture data, Python agent modules, and a documented path for future adapters.

  • Implemented Dashboard, master agent flow, data pipeline, LDA topic modeling, sentiment analysis, and Ollama response synthesis.
  • Planned Live product knowledge, pricing availability, benchmark ingestion, and richer build compatibility logic.
  • Positioning A polished open-source AI agent prototype, not a production ecommerce platform.
